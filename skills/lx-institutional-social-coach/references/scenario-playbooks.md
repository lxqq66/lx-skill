# 常见场景行动手册

## 向上沟通

1. 明确这次沟通只要信息、决定、资源还是风险知情。
2. 先给结论，再给三项以内关键事实。
3. 呈现可执行选项与取舍，不把未闭环事项包装成“都没问题”。
4. 明确责任、截止时间和下次更新节点。
5. 重要事项用合适的正式方式确认，遵守保密与记录规则。

## 接收批评

1. 先分开工作内容、对方情绪和人格化表达。
2. 能当场澄清时，询问具体例子、标准和优先级。
3. 情绪过高或公开场合不适合时，先确认任务，约定稍后复盘。
4. 对合理部分给行动和截止时间；对不实部分用事实和材料平静澄清。
5. 若存在持续羞辱、歧视或报复，转入安全与合规评估。

## 同事边界

1. 判断请求属于协作、临时互助、职责转移还是合规风险。
2. 说明当前容量和已有优先级，不编造理由。
3. 明确可做范围、所需输入和交付边界。
4. 重复越界时减少口头模糊承诺，改为适当书面确认。

## 非正式场景

1. 先确认是否自愿、是否有健康安全或利益冲突风险。
2. 可以带一个轻量目标：认识一人、了解一个项目、表达一次感谢。
3. 不靠饮酒、送礼、传播隐私或承诺不当利益换关系。
4. 工作决定回到正式流程确认。
5. 准备合理退出时间和交通安全安排。

## 政治与传言

1. 不传播无法验证的信息，不替他人判断动机。
2. 把注意力拉回任务、正式责任和可核实事实。
3. 对敏感站队邀请使用中性边界，不做虚假承诺。
4. 建立跨关系的专业可靠性，而不是押注单一人物。
5. 发现真实利益冲突、违规或安全风险时，按适当规则处理，而非私下报复。
