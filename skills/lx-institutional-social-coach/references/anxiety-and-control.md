# 社交焦虑与可控行动

## 三圆练习

```text
我能控制：
我能影响：
我不能控制：
```

从“能控制”中选择一个十分钟内可做的动作；从“能影响”中选择一次低风险沟通；对“不能控制”写一句允许不确定性存在的陈述。

## 事前两分钟

1. 双脚接触地面，呼气略长于吸气，重复数次；有身体不适则停止。
2. 写下本次唯一目标，例如“把截止时间确认清楚”。
3. 准备一句开场、一句请求、一句退出。
4. 预演三种结果：最可能、可承受的最坏、理想；每种只写一个应对。

## 暴露阶梯

从低风险、短时间、可退出的行为开始，例如：

1. 与熟悉同事进行一次两分钟工作确认。
2. 在小群提出一个事实性问题。
3. 向上级发送结构化进展。
4. 在会议中补充一个有证据的观点。
5. 进行一次边界或分歧沟通。

只有前一级能够基本承受时再提高难度。目标是带着一定紧张行动，不是等待完全不紧张。

## 事后复盘

- 实际发生了什么，而不是我猜别人怎么想？
- 哪个准备帮助了我？
- 哪个风险真实出现，哪个没有？
- 下一次只调整什么？

若焦虑持续数周并明显损害睡眠、工作或日常功能，或出现自伤想法，建议尽快联系合格专业人员；紧急危险时联系当地紧急服务和现实中的可信任支持者。
