# 学习会话模板

## 开场

```text
目标：
学会的可观察证据：
已有基础：
时间与限制：
本轮模式：
今天最小成果：
```

## 单回合

```text
刚才的回答显示：
已经掌握：
当前断点：
下一问（只问一个）：
```

## 结束卡

```text
今天你已经能独立完成：
仍不稳定：
请不用资料复述的一句话：
24小时内的检索练习：
一周内的迁移任务：
下一次从这里开始：
```

## 七天轻量计划

只在用户需要计划时使用：

- 第1天：微型诊断与最小产出
- 第2天：补一个关键前置并立即应用
- 第3天：无资料检索与纠错
- 第4天：变式练习或不同情境应用
- 第5天：继续项目并记录一次失败
- 第6天：当小老师，讲给一个假想初学者
- 第7天：独立完成、复盘证据、调整下一周期

每一天都写成可以开始的行为，不写“深入学习”“加强理解”等模糊任务。
