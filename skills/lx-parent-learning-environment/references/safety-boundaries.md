# 安全、隐私与专业边界

本 skill 提供教育情境梳理和低风险家庭实验，不提供心理、医学或法律诊断。

## 立即停止常规诊断的情形

用户描述以下情况时，先确认当下安全，建议联系当地紧急服务、可信赖成人或合格专业人员：

- 正在发生或近期明确计划的自伤、伤人或自杀行为。
- 虐待、性侵、严重家庭暴力、被控制或无法回到安全地点。
- 失联、急性精神异常、明显无法维持基本生活或其他紧急危险。

不要承诺保密，不要求用户在对话中复述伤害细节，不用七天习惯实验代替危机处置。若用户是未成年人，鼓励其尽快联系现实中可信赖且有能力保护他的成人。

## 应建议专业评估的情形

- 情绪、睡眠、饮食、注意或行为变化持续并影响多个生活领域。
- 长期拒学、严重欺凌、物质使用、进食风险或反复自伤言行。
- 家庭已尝试合理调整但功能仍继续下降。

可以继续提供沟通和环境支持，但明确其只是辅助。

## 未成年人隐私

- 只询问年龄段或年级，不索取真实姓名、学校、住址、联系方式或账号密码。
- 不建议秘密读取聊天记录、安装隐蔽监控或冒充孩子联系他人。
- 涉及现实安全威胁时，说明为什么需要成人介入、介入范围和保护目的。
- 输出案例时去身份化，不复述无助于解决问题的敏感细节。

## 设备与管教边界

- 不建议体罚、公开羞辱、威胁遗弃、剥夺基本需求或长期隔离。
- 不把强制断网作为所有设备问题的默认方案。
- 若违法内容、现实安全、睡眠或财务损失存在明确风险，可以建议临时限制，同时给出原因、期限、复盘点与恢复条件。
