# 七天家庭学习实验模板

## 设计规则

- 一次验证一个核心假设，最多改变两个环境变量。
- 目标写成可观察行为，不写“自律”“懂事”“爱学习”。
- 家长和孩子各承担一个改变，避免只改孩子。
- 保留学习结束点和恢复时间，不能把效率提升自动兑换成加量。
- 第七天根据数据继续、调整或停止，不把结果解释为人格证据。

## 输出模板

### 本周假设

我们暂时认为：当 `触发条件` 出现时，因为 `可能机制`，孩子更容易出现 `可观察行为`。若改变 `一个条件`，预计 `目标行为` 会发生可观察变化。

### 唯一目标

- 七天内希望增加或减少的一个动作：
- 发生时间与地点：
- 清晰完成标准：
- 暂时不追求的结果：

### 第 0 天：共同准备

1. 用不带责备的方式说明这是一项共同实验，而不是新的惩罚。
2. 听孩子解释最难的环节，并允许其提出一个修改意见。
3. 明确不可协商的边界和可以选择的两三个选项。
4. 调整环境，准备材料和简单记录表。

### 第 1—2 天：降低启动成本

- 把第一步缩小到五至十五分钟内可以开始。
- 由固定时间或已有生活事件触发，如“晚饭收拾完后”。
- 家长只使用约定的一次提醒；若失败，只记录发生了什么。

### 第 3—5 天：调整难度和反馈

- 若无法开始：进一步拆小、示范第一步或降低题量。
- 若过于简单：增加选择、解释、创作或挑战。
- 完成后反馈使用了什么策略、哪里比昨天顺，不增加临时任务。

### 第 6 天：让孩子多承担一步

让孩子从选择顺序、设置提醒、记录进度或发起复盘中选一项承担。成人保持边界，不突然撤掉全部支持。

### 第 7 天：共同复盘

依次讨论：

1. 哪一天最容易开始？当时有什么条件？
2. 哪个环节仍然最卡？是太难、太累、太模糊还是关系紧张？
3. 哪项改变值得保留？哪项应停止或缩小？
4. 下周只迭代哪一个变量？
