# 家庭学习干预选项库

根据已识别的主要机制选择少量措施，不把本页变成一份同时执行的清单。

## 启动困难与任务模糊

- 把“去学习”改成时间、地点、内容和结束点明确的任务。
- 把材料提前放到固定位置，减少开始前的寻找和选择。
- 设置五至十五分钟的“只开始”阶段，结束后再决定是否继续。
- 给两项真实选择，如先做计算或阅读，不给虚假选择。

## 难度失配

- 让孩子口述第一步；说不出时先示范一个相似例子。
- 将连续失败的任务缩小到可获得一次真实成功。
- 将过于简单的重复任务改成解释、找错、出题或真实应用。
- 记录卡点类型，不用单纯延长时间解决理解困难。

## 反馈过慢

- 可视化已完成步骤和剩余步骤，不只展示分数。
- 反馈策略、投入和修正，如“你换了图示后找到了关系”。
- 给任务稳定结束点；完成后不临时加量。
- 用孩子自己的前一周作基线，不用同伴作基线。

## 催促循环与关系绑定

- 约定一次提醒和一个自然后果，停止连续追问。
- 把复盘移到双方平静的固定时间，不在冲突最高点讲道理。
- 家长先承认自己的焦虑和可改变行为，不要求孩子先认错。
- 每天保留一段不谈成绩、作业和设备的共同时间。

## 电子设备冲突

- 分开记录学习、创作、社交、游戏和短视频使用。
- 先保护睡眠、用餐和高风险内容边界，再处理总时长。
- 使用共同可见的结束提示和充电位置，不秘密监控。
- 为设备承担的需要设计现实替代，如运动、同伴联系、创作和游戏化挑战。
- 设定规则试行期限与恢复信任条件，不把临时限制变成永久惩罚。

## 目标外包

- 询问孩子希望减少哪种痛苦、获得哪种能力或完成什么真实项目。
- 在成人底线内，让孩子参与选择顺序、方法和复盘指标。
- 将“为了未来”翻译成近期可以感受到的目标。
- 对年龄较小的孩子提供有限选择；随年龄与能力逐步扩大控制权。

## 精神透支

- 优先检查睡眠、身体不适、通勤、补习和连续任务总量。
- 固定无绩效要求的玩耍、运动和发呆时间。
- 若多个生活领域长期下降，不继续用效率技巧加压，转向全局诊断或专业评估。
