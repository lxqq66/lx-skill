# 家庭学习环境评估图

不必问完所有问题。根据用户已提供的信息，每轮选择二至三个最能改变判断的问题。

## 1. 事件还原

- 最近一次冲突发生在什么时候、什么地点？
- 当时孩子本来在做什么，下一项任务是什么？
- 成人原话是什么，孩子怎样回应，最后怎样结束？
- 类似情况一周出现几次，有没有例外？

## 2. 孩子状态

- 年龄或年级、近期睡眠、饮食、运动和身体不适如何？
- 在哪些活动中能持续投入？和学习情境有什么不同？
- 孩子本人怎样解释困难？他最希望成人改变什么？
- 最近是否出现持续低落、明显焦虑、拒学、自我伤害或功能下降？

## 3. 任务地图

- “完成学习”的具体终点是什么？孩子是否知道第一步？
- 任务量、预计时间和实际时间分别是多少？
- 哪个环节最容易卡住：启动、理解、坚持、检查还是收尾？
- 难度是否处于“可以开始、需要努力、看得到希望”的范围？

## 4. 环境地图

- 放学到入睡的时间线是什么？哪里最拥挤或最疲惫？
- 学习地点是否有噪声、走动、电视、食物或设备通知？
- 学习用品是否齐全，开始前要经过多少步骤？
- 成人是否在同一空间刷手机、工作、争论或频繁查看？

## 5. 反馈和关系

- 成人通常在什么时候关注学习：开始前、过程中、出错时还是完成后？
- 反馈是在提供策略信息，还是比较、催促、讽刺或人格判断？
- 孩子完成任务后会获得什么：休息、认可、继续加量，还是没有明确结束？
- 家长最担心的未来是什么？这份担心如何进入当下的对话？

## 6. 设备功能

- 设备主要用于学习、创作、社交、游戏、短视频还是混合使用？
- 最常在什么情绪、时间和场景下使用？
- 设备提供了哪些学习中缺少的体验：即时反馈、选择、成就、社交或逃离？
- 当前规则由谁制定，是否清楚、一致、可执行并有复盘期限？

## 7. 自主性和资源

- 孩子可以选择顺序、地点、方法、伙伴或休息方式中的哪些？
- 家庭中谁能稳定支持，谁的做法彼此冲突？
- 学校任务、补习、同伴关系或家庭变故是否是更大的压力源？
- 过去什么小变化曾经有效？当时有哪些条件？

## 信息足够的标准

至少拥有一个具体事件、孩子视角、成人担忧、家庭时间线和一项例外，才能形成较有把握的家庭机制判断。缺少孩子视角时，将结论标为暂定，并把“用不带指责的问题听孩子怎么说”放进实验第一步。
