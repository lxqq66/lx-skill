## 使用方法

- 每轮只选择二至三个最能减少不确定性的问题。
- 先问具体事件，再问评价；先问行为，再问动机。
- 不重复询问用户已经给出的信息。
- 敏感问题先解释目的，允许用户跳过。
- 若学生不在场，明确记录哪些内容来自成人推测。

## A. 基本情境与目标

1. 你和学生是什么关系？你最希望解决哪一个问题？
2. 学生大致年龄或年级是什么？问题从什么时候开始？
3. 请描述最近一次让你觉得“必须改变”的具体事件。
4. 三十天后出现什么可观察变化，你会认为方向是对的？
5. 学生本人认为问题是什么？他是否也希望改变？

## B. 学生当前状态

1. 最近两周睡眠、起床、食欲、精力和身体不适大致怎样？
2. 情绪和平时相比有什么变化？在哪些时段或场景更明显？
3. 最近是否经历转学、家庭变化、亲人离开、冲突、疾病或失败？
4. 在同伴、老师或家庭中，学生是否有至少一个感到安全的人？
5. 哪些活动仍能让他专注、开心或愿意主动开始？
6. 问题是否只发生在学习上，还是生活自理、社交和兴趣也受影响？

## C. 把标签还原为行为

当用户说“懒、叛逆、不自律、沉迷、玻璃心”时追问：

1. 具体做了什么或没有做什么？
2. 一周大约发生几次，每次持续多久？
3. 发生前通常有什么任务、要求、情绪或人物出现？
4. 成人怎样回应？孩子随后怎样回应？
5. 这个循环的短期结果是什么？它是否让某一方暂时轻松？
6. 有没有例外时刻？当时有什么条件不同？

## D. 学习任务和能力

1. 学生是否清楚任务的第一步和完成标准？
2. 他能独立开始，还是需要多次提醒或成人陪在旁边？
3. 真正卡住的是知识、阅读理解、书写、计划、情绪还是害怕失败？
4. 任务对他来说太难、太简单，还是数量太多？依据是什么？
5. 失败后会得到提示、重试机会，还是直接得到评价和答案？
6. 最近一次成功学习是什么？当时任务、难度和反馈怎样？
7. 学校成绩与日常理解是否一致？有没有偏科或表现反差？

## E. 自主性和目标归属

1. 在学习和生活中，学生目前能够自己决定哪些事情？
2. 他是否能说出自己想做、想学或想解决的问题？
3. 哪个目标属于学生本人，哪个主要属于家长或老师？
4. 如果没有奖励、惩罚或催促，他仍愿意做什么？
5. 他是否有从想法到小作品、行动或结果的经历？
6. 成人是否允许他犯低风险错误并参与复盘？

## F. 家庭系统

1. 家里谁最关注这个问题？不同成人的期待和规则是否一致？
2. 平时如何提醒学习？第一次提醒到发生冲突通常经历什么？
3. 家长最担心的未来后果是什么？这种担心来自什么经历？
4. 家庭规则是否具体、稳定并提前约定，还是随成人情绪变化？
5. 家长与孩子除了学习，还能否有不带教育目的的共同时间？
6. 家庭中是否有足够的睡眠、运动、自由玩耍和不被安排的时间？
7. 已经尝试过哪些办法？哪些短期有效，哪些带来副作用？

## G. 学校和课堂系统

1. 问题在所有学科都出现，还是只在某些教师、任务或时段出现？
2. 学生在课堂中有提问、选择、合作、展示或重试的机会吗？
3. 反馈主要来自分数和排名，还是也包含过程和策略信息？
4. 师生关系、同伴关系和班级安全感怎样？
5. 家校双方对问题和目标是否有相同理解？
6. 学校已经提供了哪些支持？效果如何？

## H. 手机、游戏和AI

1. 设备主要用来做什么？娱乐、社交、学习、创作还是缓解压力？
2. 最常使用的时间、地点和内容是什么？使用前通常发生了什么？
3. 结束设备使用时最容易在哪一步发生冲突？
4. 现实生活中是否有其他来源提供陪伴、掌控感、成就感和同伴联系？
5. 成人自身的设备使用是否与要求孩子的规则一致？
6. 学生使用AI时，是直接复制答案，还是会提问、验证、修改和复盘？
7. 设备问题影响了睡眠、安全、金钱或现实社交吗？

## I. 优势和保护因素

1. 学生最被认可的三项能力是什么？
2. 他在哪些场景表现出韧性、策略、照顾他人或创造力？
3. 哪位成人最能听进去他的真实想法？
4. 家庭和学校目前能稳定投入哪些时间、空间或支持？
5. 过去哪次改变最成功？可以复用什么条件？

## J. 风险筛查

仅在有相关信号时，以尊重、直接的方式询问：

1. 最近是否提到不想活、伤害自己或伤害别人？
2. 是否有人正在伤害、威胁、勒索或严重欺凌他？
3. 是否出现持续失眠、拒食、离家、失联或明显无法完成日常生活？
4. 当前是否有可信任成年人能够立刻陪伴并采取现实行动？

出现肯定回答时，转入安全分流，不继续常规教育问诊。

## 结束访谈前的核对

1. 我是否获得了一个具体事件，而不只是标签？
2. 我是否知道孩子自己的说法，或明确知道它缺失？
3. 我是否理解家长和教师各自真正害怕什么？
4. 我是否找到了至少一个例外和一个优势？
5. 我是否把主要判断写成可被七天实验验证的假设？
