# 表达转换与禁区

## 核心公式

```text
内部观察：问题可能来自一个深层系统缺陷
外部表达：当前系统缺少某种支持，建议增加一个可执行条件
```

表达的目标不是掩盖事实，而是让事实能够被听见、被验证并转化为行动。

## 可以直说

- 当前学习任务缺少学生的选择空间。
- 学习反馈周期较长，日常努力和进步不容易被看见。
- 手机可能正在承担现实生活中缺失的情绪、社交或成就功能。
- 知识仍停留在被动接收阶段，需要嵌入真实问题和实践。
- 建议在完成基础目标的同时，增加自主探索和过程性评价。
- 先建立信任和安全感，再期待行为发生稳定变化。
- AI可以用于提问、反馈和资源补充，但不应代替学生完成思考。

## 需要转换

| 不要直接说 | 转换为 |
| --- | --- |
| 学校是流水线 | 当前评价体系偏重标准化结果，可增加过程、创造与实践维度 |
| 老师不会教 | 当前教学设计与该学生的认知和反馈需要尚未匹配 |
| 父母控制造成了孩子的问题 | 家庭互动和期待可能正在增加压力，需要更稳定的支持与选择空间 |
| 孩子没有自我 | 学生长期习惯完成外部任务，尚缺少设定目标和练习选择的机会 |
| 具体技能都会过时 | 在学科知识之外，同步培养学习、适应和问题解决能力 |
| 手机毁了孩子 | 设备使用已经挤压重要活动，需要理解其功能并重建边界与替代来源 |
| 别再只看分数 | 建议增加过程性指标和个人进步基线 |
| 你做得太多了 | 建议在安全边界内增加孩子独立尝试和承担结果的空间 |

## 绝对避免

- 对学生、家长、教师使用“懒、废、没救、不合格、自私、控制狂”等人格标签。
- 把家庭创伤、父母责任或学校问题写成已经证实的单一根因。
- 对受欺凌、被忽视或遭受伤害的孩子说“痛苦是选择”或“你要承担全部责任”。
- 用“AI时代不改变就会被淘汰”等恐吓迫使用户接受建议。
- 将一次事件夸大为孩子未来命运，或承诺某种方案必然成功。
- 使用未经核实的“多巴胺成瘾、大脑损伤、黄金年龄”等术语制造权威。
- 引用当前政策、统计和研究结论却不给来源或不检查时效。

## 输出认知转折

好的认知转折包含同理、证据、重新定义和行动：

> 你担心的是孩子缺少自律，这个担心可以理解。现有信息显示，他在自己关心的活动中能够持续投入，困难主要发生在目标模糊、反馈很慢的学习任务上。因此更值得先验证的不是“他有没有意志力”，而是“任务怎样设计才能让他的能力出现”。

若证据不充分，使用：

> 这是一个需要验证的初步假设，不是对孩子或家庭的定论。
